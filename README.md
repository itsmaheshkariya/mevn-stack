# MEVN STACK
### This is Node based app based on Express framework as a server side programming and Vue will be our frontend

### Git Clone
``` git clone https://github.com/maheshkareeya/fvue-stack.git```
### Change Directory
``` cd vue-stack```
``` npm install```
```nodemon```

go to http://localhost
